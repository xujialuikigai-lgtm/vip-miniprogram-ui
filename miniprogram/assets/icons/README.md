# TabBar 图标资源

此目录存放 tabBar 所需图标文件，要求：
- 格式：PNG
- 尺寸：81px × 81px（推荐）
- 大小：≤ 40KB

## 所需图标列表

| 文件名 | 说明 |
|--------|------|
| tab-home.png | 首页未选中 |
| tab-home-active.png | 首页选中 |
| tab-category.png | 分类未选中 |
| tab-category-active.png | 分类选中 |
| tab-order.png | 订单未选中 |
| tab-order-active.png | 订单选中 |
| tab-profile.png | 我的未选中 |
| tab-profile-active.png | 我的选中 |

选中状态图标使用品牌金色 #c99a3a，未选中使用灰色 #999999。
